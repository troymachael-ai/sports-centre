# Sports Centre

## Description
This is a sports website that allows users to learn about multiple sports and register as a member of our brand. The website provides comprehensive information about different sports disciplines and a community platform for sports enthusiasts.

## Technologies
- HTML5
- CSS3
- JavaScript
- Bootstrap

## Project Structure
```
sports-centre/
├── index.html          # Home page
├── football.html       # Football sports page
├── basketball.html     # Basketball sports page
├── athletics.html      # Athletics sports page
├── tennis.html         # Tennis sports page
├── swimming.html       # Swimming sports page
├── rugby.html          # Rugby sports page
├── contact.html        # Contact & registration page
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── script.js       # JavaScript functionality
└── images/             # Sports images and visual assets
```

## Features
- Vibrant and responsive design with background colors
- Multiple sports discipline pages with detailed information
- User registration and member management
- Contact page for inquiries
- High-quality sports imagery
- Mobile-friendly layout using Bootstrap

## Getting Started
1. Clone or download the project
2. Open `index.html` in your web browser
3. Navigate through the sports pages
4. Register as a member through the contact page
